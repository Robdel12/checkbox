Checkbox

Checkbox.js aims to create a functional (and accessibile) checkbox from almost any HTMLElement.

new Checkbox(checkboxEl: HTMLElement, options: Object)
Parameters
checkboxEl (HTMLElement)
options (Object = {})
Instance Members
▸ searchForLabel()

Try to find the associated label in the DOM if no label is passed in the constructor. If the label is found it will return that element. If nothing is found it will return false.
